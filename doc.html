Hamburger Menü Beispiel

In dieser kleinen Demo könnt ihr sehen, wie man ganz ohne JavaScript ein Hamburger-Menü erstellen kann.

Außerdem seht ihr, wie Media-Queries verwendet werden können, um zwischen einem klassischen Ribbon-Menü auf dem Desktop und einem Hamburger-Menü auf Mobile zu wechseln.

Hamburger-Menüs sind in den letzten Jahren immer mehr in Verruf geraten (zu Recht, es gibt bessere Lösungen!), aber dennoch sind sie noch immer einese der häufigsten Interface-Elemente auf mobilen Endgeräten.

Dennoch solltet ihr in professionellen Web-Projekten immer die Zielgruppe im Hinterkopf behalten: versteht sie das Konzept eines Hamburger-Menüs? Oftmals ist es besser, neben den Hamburger außerdem noch das Label "Menü" unterzubringen, damit es für jeden klar wird, dass es sich um ein Menü handelt.

Öffnet diese Seite auf einem Handy, oder aktiviert in den Dev-Tools den "mobile"-Modus, um das Hamburger Menü zu sehen.

JavaScript Lösungen

Das hier präsentierte Beispiel ist eine pure CSS-Lösung, d.h. sie verwendet keinerlei JavaScript und läuft auch, wenn JS im Browser deaktiviert wurde.

Allerdings bringt das auch den Nachteil mit sich, dass das Menü z.B. nicht durch einen Klick außerhalb geschlossen werden kann und zusätzliches HTML-Markup erfordert.

Am Ende des Tages verwendet man also häufig eine hybride Lösung, oder sogar eine komplett auf JS basierende Lösung mit einem Fallback für den sehr geringen Prozentsatz an Menschen, die in ihrem Browser die Ausführung von JavaScript blockieren.

Funktionsweise

Diese Lösung basiert auf der :checked Pseudo-Klasse des <input type="checkbox">-Elements und dem "general sibling combinator" ~ aus CSS.

Wenn die Checkbox abgehakt wird, sorgt der ~ Selector dafür, dass Elemente, die danach im HTML kommen ausgewählt werden und somit spezielle Regeln auf sie angewendet werden können.

Außerdem wird die Checkbox mit einem <label>-Element verknüpft, damit sie versteckt werden kann. Ein Klick auf das Label-Element aktiviert immer das damit verbundene Input-Element.
